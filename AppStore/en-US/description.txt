Meditor is a focused, native Mermaid editor for macOS.

Write diagram source in a precise text editor and see a crisp live preview beside it. Editing, rendering, Quick Look, and export run locally, so the core workflow works without an account or internet connection.

HIGHLIGHTS

• Native .mmd and .mermaid documents with autosave, undo, and multiple windows
• Finder Quick Look previews: select a Mermaid file and press Space
• Live offline preview with pan, zoom, themes, and last-valid-preview recovery
• Syntax highlighting, line numbers, completion, and inline error navigation
• Templates for flowcharts, sequences, classes, states, ER diagrams, Gantt charts, mindmaps, architecture, and C4 context diagrams
• Rich SVG, retina PNG, and PDF export with theme and background controls
• Smart image clipboard for Slack, Notion, Keynote, and other apps
• Full-screen presentations with multi-file decks and clickable node highlights
• Optional view-only links with automatic expiry and local link management
• English and Brazilian Portuguese interface

Meditor stores diagram source as plain text and renders it locally. Diagram content is sent to the configured share service only when you explicitly choose Publish.
