Suporte

Faça o diagrama.
Ajudamos com o resto.

O Meditor é um editor Mermaid nativo para macOS 26 ou posterior.

Antes de relatar um problema

  1. Confirme que o Mac executa macOS 26 ou posterior.
  2. Teste o código usando um dos modelos incluídos no Meditor.
  3. Inclua a versão exibida em Ajustes → Sobre e Legal.
  4. Não inclua conteúdo confidencial de diagramas em relatos públicos.

Para ajuda, relatos de erros e sugestões, abra uma issue no GitHub.
