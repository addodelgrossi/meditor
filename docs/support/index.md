# Meditor Support

Meditor is a native, offline Mermaid editor for macOS.

Before reporting an issue:

1. Confirm that the Mac is running macOS 26 or later.
2. Try the same source with one of Meditor's built-in templates.
3. Include the Meditor version shown in Settings → About & Legal.

For help, bug reports, and feature requests, open a
[GitHub issue](https://github.com/addodelgrossi/meditor/issues/new).
Do not include confidential diagram content in public reports.

## Português brasileiro

# Suporte do Meditor

O Meditor é um editor Mermaid nativo e offline para macOS.

Antes de relatar um problema:

1. Confirme que o Mac executa macOS 26 ou posterior.
2. Teste o mesmo código usando um dos modelos incluídos no Meditor.
3. Inclua a versão exibida em Ajustes → Sobre e Legal.

Para ajuda, relatos de erros e sugestões, abra uma
[issue no GitHub](https://github.com/addodelgrossi/meditor/issues/new).
Não inclua conteúdo confidencial de diagramas em relatos públicos.
