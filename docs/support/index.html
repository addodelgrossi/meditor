Support

Make the diagram.
We will help with the rest.

Meditor is a native Mermaid editor for macOS 26 or newer.

Before reporting a problem

  1. Confirm your Mac is running macOS 26 or newer.
  2. Try the source with one of Meditor's built-in templates.
  3. Include the version shown in Settings → About & Legal.
  4. Do not include confidential diagram content in a public report.

For help, bug reports, and feature requests, open a GitHub issue.
