Suporte

O Meditor é um editor Mermaid nativo e offline para macOS.

Antes de relatar um problema

  1. Confirme que o Mac executa macOS 26 ou posterior.
  2. Teste o código usando um dos modelos incluídos no Meditor.
  3. Inclua a versão exibida em Ajustes → Sobre e Legal.

Para ajuda, relatos de erros e sugestões, abra uma issue no GitHub. Não inclua conteúdo confidencial de diagramas em relatos públicos.

English

Meditor is a native, offline Mermaid editor for macOS.

For help, bug reports, and feature requests, open a GitHub issue. Do not include confidential diagram content in public reports.
