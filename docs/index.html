A complete Mermaid workspace

More than a preview pane.

Meditor keeps plain-text diagrams simple while adding the native tools that make them easier to create, understand, and reuse.

Edit with confidence

Syntax highlighting, line numbers, completion, inline errors, and last-valid-preview recovery keep you moving.

Export for anywhere

Save SVG, retina PNG, or PDF, or copy a rich image that lets each destination choose its best format.

▶

Present from source

Build a temporary multi-file deck, present full screen, and highlight nodes while you explain.

Finder Quick Look

See the diagram before opening the app.

Select any .mmd or .mermaid file in Finder and press Space. Quick Look renders it locally with the same bundled engine.

Local-first architecture

Your diagram stays on your Mac by default.

Editing, rendering, Quick Look, export, clipboard, and presentations use local files and bundled resources. Publishing is an explicit, optional action.
